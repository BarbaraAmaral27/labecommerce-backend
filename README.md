## ⚙️Tecnologias utilizadas
1. ``Node.js``
2. ``Typescript.js``
3. ``API``
4. ``Express``
5. ``SQL``

##  📱 Funcionabilidades do Projeto
- [x] Cadastro de usuário:
- Id;
- E-mail;
- Password. 
<br>
- [x] Cadastro de produtos:
- Id;
- Name;
- Price;
- Category.
<br>
- [x] Cadastro de compras:
- User Id;
- Product Id;
- Quantity;
- Total Price.
